# Evaluation of collaborative methods of subsurface temperature monitoring aboard commercial fishing vessels in the Mid-Atlantic Bight
#### Description
This repository contains the data and code used to create the figures and information for Munley et al (in prep): Evaluation of collaborative methods of subsurface temperature monitoring aboard commercial fishing vessels in the Mid-Atlantic Bight. <br>

#### Authors
Mary Kate Munley, Haley Synan*, George Maynard, Anna J.M. Mercer, Sarah Salois, Kimberly Hyde <br>
*code author

## Abstract 
The Northeast U.S Continental Shelf (NES) is a dynamic and biodiverse region, supporting a number of valuable fisheries. Oceanographic observations are crucial for advancing ecosystem-based fisheries science, but historical observational programs that use gliders, buoys, and research surveys, lack the spatiotemporal resolution required for mechanistic research. Data gaps are particularly prevalent subsurface, aligning with crucial habitat for fisheries species. Oceanographic observations collected by  fishing vessels can help fill these data gaps. However, documentation on best practices for deploying oceanographic instruments on fishing vessels is limited. We compare temperature observations collected from three sensors (RBR Brevio, Lowell Instruments TDO-1, Seabird 19+) and deployed using three methodologies (headrope, doors, winch) during commercial fishing trips in 2024 and 2025. Temperature observations were comparable across all methodologies and sensors, although each requires varying degrees of technician involvement and fishing crew labor. Here we establish a framework for instrument selection and deployment methodology to support the continued expansion of cooperative ocean observing. 
## Repo set up 
* ```/data```: contains raw data (csv) from the evaluated sensors
* ```/code```: contains reproducible code to process, visualize, create the figures in the short communication.
